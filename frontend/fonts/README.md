# Fonts

Đặt các file font vào thư mục này:

- `AndadaPro-Regular.ttf` - Cho text "Chào, Anh Duy" và "ver: 1.0.0"
- `AlumniSans-Regular.ttf` - Cho text "Chưa có thông báo nào!"

## Download fonts

1. **Andada Pro**: https://fonts.google.com/specimen/Andada+Pro
2. **Alumni Sans**: https://fonts.google.com/specimen/Alumni+Sans

Sau khi download, đặt file `.ttf` vào thư mục này.

## Lưu ý

Nếu chưa có fonts, app sẽ sử dụng system fonts làm fallback.
